Описание: Создайте веб-приложение для управления задачами, где пользователь может добавлять, отмечать выполненные и удалять задачи.

Функционал:

Главная страница: Пользователь видит список своих задач. Каждая задача включает заголовок, описание, дату создания и статус выполнения.

Создание новой задачи: Пользователь может добавить новую задачу, указав заголовок и описание.

Отметка выполнения: Пользователь может отметить задачу как выполненную/невыполненную. Статус выполнения меняется на основе действий пользователя.

Редактирование задачи: Пользователь может редактировать заголовок, описание и другие детали задачи.

Удаление задачи: Пользователь может удалить задачу, при этом должно быть подтверждение для предотвращения случайного удаления.

Фильтрация задач: Пользователь может фильтровать задачи по статусу выполнения (все/выполненные/невыполненные).

Поиск задач: Добавьте функцию поиска, чтобы пользователь мог находить задачи по ключевым словам.

Сохранение данных: Данные о задачах должны сохраняться в Redux Store (или локальном хранилище браузера).

React Router: Используйте React Router для навигации между страницами (список задач, создание/редактирование задачи).

Дополнительные возможности: Вы можете добавить дополнительные функции, такие как приоритеты задач, категории, установку напоминаний и т.д.
